File SPPT Anda Terenkripsi

SPPT_PBB_2026.xlsx terdeteksi berpassword. Ikuti langkah ini di Excel:

  1. Buka file di Microsoft Excel
  2. Klik File → Info → Protect Workbook → Encrypt with Password
  3. Hapus password (kosongkan) → OK
  4. Simpan ulang (Ctrl+S)
  5. Upload file yang sudah tidak berpassword
